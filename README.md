# influxdb-sample

## Usage

```commandline
python main.py -h
usage: main.py [-h] [-w] -c COUNT -p {s,ms,us,ns}

optional arguments:
  -h, --help            show this help message and exit
  -w, --write           write
  -c COUNT, --count COUNT
                        count, 0 for non-stop
  -p {s,ms,us,ns}, --precision {s,ms,us,ns}
                        time precision
```
